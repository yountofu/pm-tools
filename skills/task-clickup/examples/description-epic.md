# [Epic] 티켓 Description 작성 예시

> [Epic] 비밀번호 재설정 태스크에 작성된 예시

---

## 기능 요약

비밀번호 재설정 — 비밀번호를 잊은 사용자가 이메일 인증을 통해 새 비밀번호를 설정한다.

## 접근 경로

| 항목 | 값 |
|------|-----|
| URL | `/reset-password` |
| 접근 방법 | `/login` → '비밀번호를 잊으셨나요?' 클릭 |
| 대상 사용자 | 비로그인 사용자 |

## 주요 기능

| 기능 | 설명 | 우선순위 | MVP |
|------|------|----------|-----|
| 이메일 입력 | 가입한 이메일로 계정 확인 | P0 | Y |
| 이메일 인증 | 인증 코드로 본인 확인 | P0 | Y |
| 새 비밀번호 설정 | 비밀번호 규칙에 맞는 새 비밀번호 입력 | P0 | Y |
| 완료 후 로그인 안내 | 재설정 완료 후 로그인 페이지로 연결 | P0 | Y |

## 사용자 흐름

```mermaid
flowchart LR
    A[로그인 페이지] --> B[비밀번호 재설정 클릭]
    B --> C[이메일 입력]
    C --> D[인증 코드 입력]
    D --> E[새 비밀번호 설정]
    E --> F[재설정 완료 → 로그인 이동]
```
